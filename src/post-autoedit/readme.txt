Plugin for automatically editing post titles.

== Description ==

This plugin automatically edits post titles. Recursively do operations like changing capitalization to all post titles under a certain type, or even by specifying post IDs. Compatible with other major plugins.

Features:

* Backup current post titles;
* Import post titles;
* Mass edit all posts under a certain post type;
* Edit specific posts by ID;
* Extremely lightweight.

You can check out the source code, submit patches, improvements or translations, and report bugs at the project's GitHub page: https://github.com/cg-alves/CGA-Post-AutoEditor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/cga-post-autoedit` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Select "Plugin Helper" from the sidebar
4. Select your desired method for installing plugins
